Data Sturctures & Collections

1. Fundamental DT
    A variable will store only single data

2. Collective DT
    1. list []
        eg. [2, 2.5, "abc", True] - multiple data type
        -> Ordered (it contain indexes)
        -> Mutable
        -> Duplicates is allowed

        Methods:
            1. to acess an element we can do it by list_name[index]
            2. to add element
                1. append
                    list_name.append(value)
                    list_name.append(value, value, value ...)
                        -> if multiple value it will create nested list_name
                2. insert
                    list_name.insert(index, value)
                    -> existing value will shift right
                3. extend
                    list_name.extend(value)
                    list_name.extend(value, value, value ...)
                    -> multiple value at a same time
            3. to remove an element
                1. remove 
                    list_name.remove(value)
                2. pop
                    list_name.pop()
                    -> remove last element
                3. del
                    del list_name[3]
                4. clear
                    list_name.clear()
                    -> remove all element inside the list_name

        list Comprehension
            eg. list_name = (i for i in range(1, 6))
            print(l) -> (1, 2, 3, 4, 5)

            print([i for i in range(1, 50) if i % 2 == 0])

            l1 = list(
            i for i in range(1, 50)
            if i % 2 == 0
            )
            print(l1)

            l = [[1, 2, 3], [4, 5, 6]]
            print(l[1][2])

    2. Tuple ()
        eg. tuple_name = (1, 2, 3, 4, 5)
        -> Immutable

        Methods:
            1. count
                tuple_name.count()

        Tuple Comprehension
            t1 = tuple(i for i in range(1, 6))

    3. set {}
        eg. set_name = {1, 2, 3, 4, 5}
        -> unique element
        -> Mutable
        
        Methods
            1. add element
                set_name.add(value)
            2. remove element
                1. set_name.remove(value)
                2. set_name.pop()
            3. .
                1. union
                    print(set_name_a | set_name_b)
                    -> all element
                2. intersection
                    print(set_name_a & set_namee_b)
                    -> same element
                3. difference 
                    print(set_name_a - set_name_b)
                    -> different element

        set Comprehension
            set1 = set(
                i for i in range(1, 6)  
            )

    4. Dictionary {key : value}
        eg. dict = {
            "name" : "NIMS",
            "estd" : 2000,
            "course" : "B-Tech"
            }

        Methods
            1. .key
                print(dict.keys())
            2. .get
                print(dict.get("estd"))
        
        dict Comprehension
            d1 = {
                i : i*i for i in range(2, 10)
            }
            print(d1)
